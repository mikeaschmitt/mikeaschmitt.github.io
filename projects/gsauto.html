GS Automated

Description

This program is a simple GUI that allows a reviews team for PC performance

testing teams to automate the process of running individual tests rather

than run them manually. The program saves the users a few days of work by running

overnight and finishing the next day. After the tests complete the information

is stored into an excel document with another tab of the program.

Here is the master screen to configure the tests to be run as well as the

number of times the tests would be run. There is also a place to output

the resulting information

This is the tab that allows the information to be exported to an excel doument.
